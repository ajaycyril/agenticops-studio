# Edge AI Production Path

Edge and cloud should work together. Edge nodes handle perception, filtering, buffering, and first-level response. Cloud control planes coordinate SOP, policy, approvals, fleet management, and audit records.

Production concerns:

- Local camera inference.
- Edge gateway identity and health.
- Model packaging and rollout rings.
- Offline mode and event buffering.
- Cloud sync and reconciliation.
- Versioned model and policy bundles.
- Device health and tamper detection.
