# Product Requirement: AgenticOps Studio

AgenticOps Studio is a Vercel-deployable Next.js application that teaches Physical AI, Edge AI, and governed enterprise agentic AI through a Hassantuk-inspired fire-response workflow.

## Thesis

Rule-based automation detects. ML predicts. Agentic AI coordinates. Enterprise agentic AI governs execution.

## Core Scenario

Smoke is detected in a smart villa/community. Sensors trigger, camera or edge vision analyzes smoke/fire, a risk model predicts fire probability, and an agentic control plane gathers evidence, checks SOP, evaluates policy, requests human approval, executes safe sandbox actions, and writes a decision record.

## Requirements

- Interactive scenario simulator.
- Rule vs ML vs Agentic comparison.
- Roboflow vision route with fallback.
- TensorFlow.js browser training.
- OpenAI structured-output agent route with fallback.
- Policy evaluator and approval gates.
- Tool registry and sandbox executor.
- Trace viewer and decision records.
- Architecture and enterprise blueprint pages.

## Non-Goals

No real emergency dispatch, no real access-control unlock, no real drone SDK integration, no external database requirement, and no Kubernetes requirement for the public demo.
