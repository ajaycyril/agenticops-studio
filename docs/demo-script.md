# Demo Script

## 3-Minute Demo

1. Open Studio and load Confirmed fire.
2. Run rule-based automation and show fixed threshold logic.
3. Run ML prediction and explain probability without execution.
4. Run vision model and show Roboflow/sample fallback.
5. Run agentic planner and show proposed actions.
6. Approve or reject a gated action.
7. Write and download a decision record.

## 8-Minute Demo

1. Start with the thesis: rule-based detects, ML predicts, agentic AI coordinates, enterprise AI governs.
2. Compare false alarm, confirmed fire, low confidence camera, degraded sensor, and drone unavailable presets.
3. Train a browser model and promote a model version.
4. Run vision with sample or uploaded image.
5. Explain each logical agent node.
6. Walk through policy decisions and human approval.
7. Inspect trace events.
8. Generate the decision record and architecture page.
